# TREEMOR Jupyter Notebooks

This directory contains Jupyter notebooks for TREEMOR analysis, validation, and tutorials.

## Notebook List

| Notebook | Description |
|----------|-------------|
| `01_fsin_calculation.ipynb` | Introduction to FSIN parameters and calculations |
| `02_resonance_validation.ipynb` | Validation of cantilever beam resonance model |
| `03_coupling_analysis.ipynb` | Root-soil coupling analysis across soil types |
| `04_event_detection.ipynb` | Seismic event detection with tree networks |
| `05_machine_learning.ipynb` | ML classifiers for event discrimination |
| `06_case_studies.ipynb` | Case studies: Noto Peninsula, Vancouver Island |
| `07_tssi_optimization.ipynb` | Optimizing TSSI for site selection |
| `08_infrasound_detection.ipynb` | Atmospheric infrasound detection |
| `09_satellite_insar.ipynb` | InSAR satellite canopy monitoring |
| `10_deployment_planning.ipynb` | Global deployment planning tool |
| `11_validation_847_events.ipynb` | Full validation on 847 events |
| `12_global_network_simulation.ipynb` | Global forest network simulation |

## Usage

```bash
# Install Jupyter
pip install jupyter

# Launch Jupyter Lab
jupyter lab

# Or Jupyter Notebook
jupyter notebook
```

Requirements

· Python 3.9+
· Jupyter
· treomor package installed
