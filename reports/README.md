# TREEMOR Reports Directory

## Structure

| Directory | Description | Format |
|-----------|-------------|--------|
| `daily/` | Daily monitoring reports | .txt |
| `weekly/` | Weekly summary reports | .txt |
| `monthly/` | Monthly analysis reports | .txt |
| `alerts/` | Event alerts and warnings | .txt |

## Report Format

All reports are plain text (.txt) format for compatibility and simplicity.

## Auto-Generation

Reports are automatically generated by the monitoring system.

## Retention

- Daily reports: 30 days
- Weekly reports: 12 weeks
- Monthly reports: 12 months
- Alerts: Permanent
